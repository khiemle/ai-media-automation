# Channel Presets

Default scene parameters for the user's two YouTube channels. When the user names a channel, pre-fill these and let them override per step. Source: Channel_Launch_Plan_ASMR.md and Channel_Launch_Plan_Soundscapes.md.

---

## ASMR Sleep & Relax

Audience is trying to fall asleep or wind down at night. The visual must support sleep — anything visually exciting is a defect. Most viewers will play the video with their eyes closed within 2 minutes, so the *first impression* matters most for thumbnail click, and the *looping content* must be safe to stare at if they don't fall asleep immediately.

| Field | Default |
|---|---|
| **Use case** | Sleep, deep relaxation, stress relief |
| **Video length** | 8–10 hours |
| **Mood** | Dark, intimate, hushed, slightly melancholic |
| **POV (Step 2)** | Inside-out through window (default) — anchors viewer in a warm interior |
| **Time of day (Step 3)** | Night, late evening, or 3am storm |
| **Weather (Step 3)** | Heavy rain / thunderstorm / snowfall / fog. Some weather is essential. |
| **Atmosphere (Step 3)** | Slight haze, condensation on windows, occasional distant lightning |
| **Style family (Step 4)** | Photoreal / cinematic (default). Painted is okay for fantasy themes. Anime rarely fits. |
| **Light direction (Step 5)** | Single warm practical light from inside (lamp, fire) + cool ambient from outside (moon, streetlight). Strong rim lighting on raindrops/condensation. |
| **Color palette (Step 5)** | Deep navy `#0D1B2A` + charcoal + warm amber accent `#C8956C`. Very low saturation. Background near-black `#080C12`. |
| **Depth of field (Step 5)** | Shallow — foreground (window/raindrops) sharp, background (city, storm) blurred. |
| **Motion strategy (Step 7)** | Static camera. Cinemagraph-style loop. |
| **Motion intensity (Step 7)** | 1–2 / 10. Barely-perceptible movement. |
| **Ambient motion elements (Step 7)** | Rain droplets running down glass at varied speeds, very slow drift of fog/steam, gentle candle flicker, occasional distant lightning flash (rare — once per minute max). |
| **Anti-patterns** | Bright colors. Sudden visual events. Fast camera moves. People in frame. High saturation. Anything cartoonish. |

### ASMR theme cheatsheet

These map to the `Tier 1 / 2` themes from the channel plan:

| Theme | Notes |
|---|---|
| Rain on window (bedroom) | Warm interior, dark city beyond, raindrops running down glass. Hero theme. |
| Thunderstorm | Same as rain-on-window but with rare distant lightning flashes — keep them rare so they don't wake the viewer. |
| Ocean waves | Open POV variant (D — floating). No interior. Predawn or moonlit. Very long horizon. |
| Dark screen rain | Near-black image with only window-frame silhouette and tiny rain streaks visible. Extreme low-light. |
| Fireplace + rain | POV variant C (cozy interior). Hearth as light source, rain hinted through window. |
| Library at night | POV variant C. Tall shelves, single desk lamp, rain barely audible on a high window. |

## Soundscapes & Ambience

Audience is awake and working / studying / meditating. Visuals can be more vibrant and engaging — they're meant to be glanced at occasionally, not avoided. Daylight scenes are common. Subtle melodic music is in the audio track, so visuals can match more energy.

| Field | Default |
|---|---|
| **Use case** | Study, focus, work-from-home, meditation |
| **Video length** | 2–4 hours |
| **Mood** | Inspiring, immersive, "step into another world" |
| **POV (Step 2)** | Outside in landscape (default for nature themes). Cozy interior for café/library themes. |
| **Time of day (Step 3)** | Golden hour (sunrise / sunset) or soft midday for nature. Night for urban/Tokyo themes. |
| **Weather (Step 3)** | Clear with mist, light rain, soft fog, gentle wind. Storms are rare. |
| **Atmosphere (Step 3)** | Volumetric god-rays through trees, drifting fireflies, blossom petals, distant mist. |
| **Style family (Step 4)** | Photoreal / cinematic (default). Anime/Ghibli excellent for forest, garden, fantasy themes. Painted for "wizard's library", "hobbit hole". Surreal for "underwater", "space station". |
| **Light direction (Step 5)** | Single strong directional from sun/window-light + warm bounce. Backlit subjects with rim light are signature look. |
| **Color palette (Step 5)** | Forest green `#2D4A3E` + sky blue `#7FB3C8` + golden hour accent `#E8B84B`. Higher saturation than ASMR but still painterly. |
| **Depth of field (Step 5)** | Medium — foreground readable, far background softly blurred. |
| **Motion strategy (Step 7)** | Static camera (default) OR very slow imperceptible push-in / push-out (5 seconds total range). Never both. |
| **Motion intensity (Step 7)** | 3–4 / 10. Gentle ambient motion is okay. |
| **Ambient motion elements (Step 7)** | Foreground leaves sway, water surface ripples, mist drifts, fireflies float, light rays shimmer, distant blurred figures move (cities), curtains breathe. |
| **Anti-patterns** | Static photo with no motion (looks dead). Fast pans. Time-lapse motion. Multiple competing motion centers. People in sharp focus. |

### Soundscapes theme cheatsheet

| Theme | Notes |
|---|---|
| Forest stream | Outside POV. Misty morning, sunlight through trees, foreground moss. Hero theme. |
| Mountain waterfall | Outside POV. Wide shot, distant fall, foreground mossy rocks. |
| Bamboo forest | Outside POV. Anime/Ghibli style works beautifully. Wind moves bamboo tops only. |
| Tokyo night | Inside-out POV (apartment) OR street-level outdoor (no POV anchor). Neon, wet pavement, distant blurred pedestrians. |
| Cozy library | Cozy interior POV (C). Warm candle/lamp + tall shelves. Painted style fits well. |
| Beach sunset | Outside POV. Wide horizontal, gentle waves, golden hour. |
| Underwater | Floating POV (D). Coral, slow particles, god-rays from above. Surreal style fits. |
| Hobbit hole | Cozy interior POV (C). Round window looking out at green hills. Painted/Ghibli. |

## Custom / Other channel

If the user is producing for a different channel or just experimenting, don't apply either preset. Run the full interview from scratch but volunteer parallel options from both presets if the theme overlaps.
