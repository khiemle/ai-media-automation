## 10. Automation Script — Hướng dẫn sử dụng

<!-- Automation script section: reference the pipeline.py CLI commands specific to this channel.
     Include real example commands with --channel set to this channel's identifier. -->

File: `pipeline.py` (xem file riêng trong cùng thư mục)

### Cài đặt

```bash
pip install rich click colorama
python pipeline.py --help
```

### Các lệnh chính

#### Tạo metadata cho 1 video
```bash
python pipeline.py generate \
  --channel [channel-id] \
  --theme "[example theme]" \
  --length [Xh] \
  --date [YYYYMMDD]
```

<!-- channel-id: short slug used in filenames and CLI, e.g. "asmr" or "soundscapes" -->

#### Quản lý upload queue
```bash
python pipeline.py queue add \
  --channel [channel-id] \
  --file [filename].mp4 \
  --date [YYYY-MM-DD] \
  --time [HH:MM]

python pipeline.py queue list
python pipeline.py queue done --id [N]
```

#### Đặt tên file hàng loạt
```bash
python pipeline.py rename --folder ./raw_exports --channel [channel-id]
```

#### Xem content calendar
```bash
python pipeline.py calendar --channel [channel-id] --month [YYYY-MM]
```
