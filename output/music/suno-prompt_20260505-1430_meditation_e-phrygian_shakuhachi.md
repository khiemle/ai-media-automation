**Step 2 — Extend in Suno to ~60 min unique audio**

**Step 3 — Loop in post-production**  
For a 1–3 hour YouTube video, loop the 60-min unique audio in your DAW or via ffmpeg:

```bash
# Example: 3-hour output from a 60-min file
ffmpeg -stream_loop 2 -i mountain_meditation_60min.wav \
  -af "afade=t=in:st=0:d=3,afade=t=out:st=10797:d=3" \
  -c:a pcm_s16le mountain_meditation_3hr.wav
```

Apply a 3-second crossfade at each loop join. Meditation listeners are inward-focused — the loop seam is undetectable with a loop-safe seed.
