Pivot từ mô hình cover sang **nhạc gốc do AI tạo** (Suno/ElevenLabs):

## 4. Đặc điểm âm thanh

**Nguyên tắc âm thanh:**
- Lofi acoustic / chillhop — CÓ melody, CÓ groove (khác hẳn ambient drone). Melody mềm, không lấn, loop-friendly.
- BPM 70–90, key thường minor hoặc major ấm (A minor, C major, D minor, F major)
- Instrumentation: felt piano hoặc nylon guitar lead + upright bass + brushed drums + ambient pad + vinyl crackle
- Chord vocabulary: jazzy — maj7, min9, extended chords (signature lofi)
- Performance feel: behind-the-beat groove, humanized (không quantize cứng) — đây là authenticity lever lớn nhất
- Vinyl crackle layer rất nhẹ, warm tape saturation trên master
- 100% instrumental — Suno Instrumental toggle ON, không vocal
- Master target -14 LUFS (Spotify/YouTube standard)

**Volume mix (trong 1 track):**
```
Lead (felt piano / nylon guitar): 0dB reference
Upright bass: -6dB
Brushed drums: -9dB
Ambient pad: -16dB
Vinyl crackle bed: -22dB
Tape hiss / room tone: -26dB
```

**Compilation strategy:** mix 1 giờ ≈ 15–20 track riêng biệt cùng series identity, crossfade vào nhau. KHÔNG loop 1 track 4 phút × 15. Mỗi track có intro mềm + ending không cắt đột ngột để crossfade sạch.

## 8. Production Pipeline — Step by Step

### Tổng thời gian mỗi mix 1 giờ: 90–150 phút (gồm batch 15–20 track)

#### Bước 1 — Chọn theme & brief (10 phút)
```
1. Chọn theme tuần này từ Theme Library (Section 3)
2. Xác định series identity: genre lofi, key family, tempo band, instrumentation, mood
3. Đây là input cho skill music-video workflow Music
```

#### Bước 2 — Generate music data qua skill music-video (15 phút)
```
1. Gọi skill: music-video → workflow Music → genre lofi
2. Skill chạy 2-persona interview (Producer + lofi Artist) cho SERIES IDENTITY
3. Skill xuất: Suno Style/Title/Exclude + ElevenLabs fallback + Producer/Artist notes
4. File lưu: working/{theme-slug}/json/ + md/
5. Với mix 1h: lặp lại phần config nhẹ cho 15–20 track variation (giữ identity, đổi melody/mood)
```

#### Bước 3 — Generate nhạc trên Suno (40 phút cho batch 15–20 track)
```
1. Mở Suno → Custom Mode, Instrumental ON
2. Paste Style of Music + Title + Exclude Styles từ skill output
3. Generate 2–4 variation mỗi track → chọn bản tốt nhất
4. Nếu Suno fail/rate-limit → fallback ElevenLabs Music với prompt từ skill
5. Download WAV tất cả track
6. Đặt tên: lofi_{theme-slug}_{NN}.wav
```

#### Bước 4 — Compile + master (Audacity/Audition, 25 phút)
```
1. Import 15–20 track vào timeline
2. Sắp xếp theo mood arc (mở đầu nhẹ → giữa → kết lắng)
3. Crossfade 2–4s giữa các track
4. Master bus: tape saturation nhẹ, LUFS normalize -14
5. Export WAV mix 1h: lofi_{theme-slug}_MIX.wav
```

#### Bước 5 — Generate visual qua skill music-video (20 phút)
```
1. Gọi skill: music-video → workflow Visual → genre lofi
2. Skill xuất Midjourney prompt + Runway prompt + creative brief
3. Midjourney: generate → upscale → lofi_{theme-slug}_visual.png
4. Runway Gen-4: upload PNG + Runway prompt → motion 2-3/10 → loop 5s
5. Download: lofi_{theme-slug}_loop.mp4
```

#### Bước 6 — Ghép video (CapCut, 20 phút)
```
1. New project 1920×1080, 24fps
2. Import: visual_loop.mp4 + MIX.wav
3. Loop visual đến khớp audio length, crossfade 0.5s
4. Fade-in 1s, fade-out 3s
5. Export: 1920×1080, 24fps, 16Mbps, MP4 H.264
6. Tên: lofi_{theme-slug}_FINAL.mp4
```

#### Bước 7 — Generate SEO qua skill music-video (10 phút)
```
1. Gọi skill: music-video → workflow SEO → genre lofi
2. Skill đọc paired music + visual JSON, xuất: title + 4 alt, description, 40 tags, 3 thumbnail text, pinned comment, chapters
3. File lưu: working/{theme-slug}/json/ + md/
```

#### Bước 8 — Thumbnail + Shorts cut (15 phút)
```
1. Thumbnail Canva: visual.png + text từ skill SEO output
2. Shorts: cắt 60s đoạn hay nhất từ MIX → vertical 1080×1920 + text overlay
```

#### Bước 9 — Upload & Schedule (10 phút)
```
1. YouTube Studio → Upload FINAL.mp4
   - Title, description, tags, chapters từ skill SEO output
   - Thumbnail, playlist
   - ✅ Tick "Altered or synthetic content" (AI disclosure)
   - Schedule theo content calendar
2. Pin comment từ skill output
```

### Batch Production (Khuyến nghị)

```
Thứ 2 (3h): Skill music-video — generate music data cho 3-4 theme tuần này
Thứ 3 (3h): Batch Suno generation — tất cả track
Thứ 4 (3h): Batch compile + master tất cả mix
Thứ 5 (3h): Skill music-video — visual + SEO; Midjourney + Runway; CapCut ghép
Thứ 6 (1h): Thumbnail + upload schedule cả tuần
```

**Tổng thời gian/tuần:** ~13 giờ cho 3–4 video + Shorts
